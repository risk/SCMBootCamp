コマンド 説明 使用例
branch ブランチの一覧を表示 git branch
branch ブランチを新規作成 git branch ${ブランチ名}
branch ブランチを削除 git branch -d ${ブランチ名}
clone リポジトリの複製（クローン）を作成します。 git clone ${複製するリポジトリのURL} ${opt-ローカルのディレクトリ名}
branch ブランチを削除(未マージのブランチを強制削除) git branch -D ${ブランチ名}
branch 各ブランチにおける直近のコミットを調べる git branch -v
branch 現在のブランチにマージ済みのブランチを調べる git branch --merged
branch 現在のブランチに未マージのブランチを調べる git branch --no-merged
push 指定のリポジトリに自分のリポジトリのデータを送信する。 git push ${送信先リポジトリ} ${送信元ブランチ}:${送信先ブランチ}
reset HEADの位置を1つ前に戻す。インデックスとワーキングツリーには影響は与えない git reset --soft HEAD^
reset HEADの位置、インデックスを2つ前に戻す。ワーキングツリーには影響は与えない git reset --mixed HEAD^^
reset HEADの位置、インデックス、ワーキングツリーを3つ前の状態に戻す git reset --hard HEAD^3
commit コミットを実行する git commit
commit オプションで変更されたファイルを自動検出してコミットする。 git commit -a
commit オプションでコメントを引数として指定できる。 git commit -m
commit 指定したファイルのコミットを実行する git commit ${コミットするファイル名}
stash ステージされていない変更を一時的に保存する
行うごとに蓄積していく		 git stash
git stash save ${メッセージ}
stash list 保存したリストを確認する git stash list
stash pop 一番最近に保存した変更を元に戻す git stash pop
stash drop 一番最近に保存した変更を捨てる git stash drop
stash clear 保存した変更をすべて捨てる git stash clear
add 任意のファイルをコミット対象として追加する git add ${コミット対象として追加するファイル名}
add 全てのファイルをコミット対象として追加する git add .
git pull ほかのリポジトリの変更点をローカルリポジトリにマージする git pull ${変更点の取り込み元リポジトリ}
checkout ブランチを切り替える git checkout ${ブランチ名}
checkout ブランチを作成して、そのブランチに切り替える git checkout -b {ブランチ名}
git merge 現在のブランチに指定したブランチへのコミットをマージする git merge ${ブランチ名}
status 変更が加えられたファイルを表示する git status
status 変更が加えられたファイルを表示する（短縮形で表示） git status -s
rm 指定したファイルを、ツリーから削除する git rm ${削除するファイル名}
log コミットログを表示する git log
log コミットログを表示する git log
log コミットログと変更の状態を表示する git log --stat
log コミットログを指定した数だけ表示する git log -n 5
log 特定のファイルのコミットログを表示する git log file.txt
log 著者、コミッタを正規表現で検索してログを表示する git log --committer=regexp
log 指定した時間に当てはまるログを表示する git log --since=2011-2-6
log コミットメッセージを逆順にして表示する git log --reverse
mv ファイルを移動する git mv ${from} ${to}
fetch リモートリポジトリから最新のコミットを取得する(pullと異なりマージは行わない) git fetch ${リポジトリ名} 又は git fetch ${URL}
help コマンドの一覧を表示 git help
help -a 利用可能なコマンドとパラメータの一覧を表示 git help -a
help ${command} ${command}で指定されたコマンドの詳細ヘルプを表示 git help ${command}
help -m ${command} ${command}で指定されたコマンドの詳細ヘルプをman形式で表示 help -m ${command}
help -i ${command} ${command}で指定されたコマンドの詳細ヘルプをinfo形式で表示 help -i ${command}
help -w ${command} ${command}で指定されたコマンドの詳細ヘルプをhtml形式で表示 htlp -w ${command}
