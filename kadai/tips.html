HTTPのパスワード githubにHTTPアクセスすると毎回のアクセスの時にパスワードが聞かれます。これを設定ファイルに書いておくと毎回の入力を省略できます。
Linux/Mac OS X
[~/.netrc]

machine github.com
login <githubユーザ名>
password <githubパスワード>

Windows
[%HOME%¥_netrc]

machine github.com
login <githubユーザ名>
password <githubパスワード>
